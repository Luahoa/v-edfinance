# V-EdFinance Design System

**Version:** 1.0  
**Status:** Foundation Phase  
**Frameworks:** Tailwind CSS + shadcn/ui

## Philosophy
**"Farming Prosperity"**
- **Growth:** Green primary color symbolizes rice fields and financial growth.
- **Trust:** Clean, accessible typography and structure.
- **Urgency:** Red accents for "Loss Aversion" (Nudge theory).

## Core Systems
1. [**Colors**](./01-colors.md) - Green/Blue/Red semantic palette.
2. [**Typography**](./02-typography.md) - Inter font, financial data formatting.
3. [**Spacing & Layout**](./03-spacing.md) - 4px grid, container strategies.
4. [**Components**](./04-components.md) - Component library audit (In Progress).
5. [**Patterns**](./05-patterns.md) - Behavioral finance UI patterns (In Progress).
